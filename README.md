IPL Data Analytics Dashboard

Project Overview
This project is an end-to-end IPL Data Analytics Dashboard developed using SQL and Power BI. The objective of this project is to analyze IPL match and ball-by-ball datasets to generate meaningful insights related to player performance, team success, toss impact, venue trends, and match strategies.

The project demonstrates practical data analytics skills including SQL querying, data aggregation, dashboard design, and data visualization.


Tools & Technologies Used
- MySQL Workbench
- Power BI
- CSV Dataset
- Kaggle IPL Dataset


Dataset Information
The dataset contains:
- IPL Match-Level Data
- Ball-by-Ball Delivery Data
- Batting Statistics
- Bowling Statistics
- Toss Results
- Venue Information

Dataset Source:
https://www.kaggle.com/datasets/ramjidoolla/ipl-data-set
